Tuplain User's Guide

Tuplain is a simple "YouTube doubler" that allows you to play back two videos at the same time. Unfortunately it has no proper user interface so you must manipulate the URL by hand.

Create an URL with the following parameters:

  • audio = YouTube ID for the audio track (also a)
  • video = YouTube ID for the video track (also v)
  • audiostart, videostart = start time in seconds (also as, astart, vs, vstart)
  • vrate, arate = playback speed. Try 0.25, 0.5, 1.0, 1.5 or 2.0.
  • vvol, avol = volume in range 0-100. (also vv, av)
  • len = length in seconds (also l)

For example:

?audio=tq4tr6fWnsA&video=bmxLE9RDPk4&astart=50&vs=22&vrate=0.5
